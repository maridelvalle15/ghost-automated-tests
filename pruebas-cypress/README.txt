Instrucciones para correr la prueba

1. Abra Cypres y para iniciar el proceso elija la carpeta Integration 
2. Desde Cypres entre la la carpeta examples y dé clic en el archivo denominado monkey_testing.spec.js
3. La prueba comenzará a correr y le mostrará el resultado
4. La prueba selecciona aleatoriamente entre 4 tipos de pruebas distintas
5. Tenga en cuenta que hay una prueba que intenta obtener el tag 'select' pero este tag no existe en la página que se está probando, por lo que mostrará un error. 
5. Si muestra error, pause la prueba y vuelva a correrla
